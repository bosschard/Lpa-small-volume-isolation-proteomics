# Small Volume Lipoprotein (a) Isolation Proteomics


### Pamir Lab, Knight Cardiovascular Institute, Oregon Health & Science University

Source file and R markdown containing statistical analysis and figures for the manuscript **Small Volume Lipoprotein (a) Isolation Guide for Clinical Research** (Mueller et al).
This dataset is the result of data-dependent acquisition (DDA) mass spectrometry proteomics investigating the proteomes of Lp(a), HDL, LDL, and VLDL.
